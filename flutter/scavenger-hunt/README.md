I did an android application using flutter about a scavenger hunt that shows various spots in the city I live in. Those spots have a special meaning for my wife and I.

It uses geolocalisation to check if the user is under a certain amount of meters from the destination. If it is, then it gives a clue for the next location as well as the distance the user is from it.

I also implemented the possibility to skip a destination, in case we ran out of time.

The code is in another repository that is private.
